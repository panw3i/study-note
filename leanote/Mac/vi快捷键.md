```
ctrl + p   切换文件，选择文件，搜索文件

ctrl + u  定位到第一行

ctrl + d 定位到最后一行

ctrl + e 显示左侧文件面板

ctrl + n  进入选择模式，可以移动光标选择，选择后可以一次性删除或者修改

双击

o 打开目录和文件
go 预览
t 打开新标签页
T 单独窗口打开一个新的tab页

鼠标中间
i 打开左侧文件目录
gi 预览
s   在底部打开文件目录
gs  预览文件目录

单击
o  打开关闭节点
x  关闭目录

菜单选择
P    直接选择root
p   选择父目录
K   选择一个字文件或者目录
J   选择最后一个文件或者目录
ctrl + k 选择上一个文件
ctrl + j 选择下一个文件
h j k l  上下左右移动光标

文件映射
r   刷新目录
m   显示菜单
cd  改变目录
CD  直接返回到根目录

q   退出目录
A   放大或缩小
？  显示帮助


za  展开当前行折叠的代码
zR  展开所有折叠的代码

,tt,  关闭打开的tab页面

格式化代码，选中代码后

<  <<
或者
> >>
```
